# lux
DB + UI multiplataforma para gestão de bibliotecas de casas espíritas, que tenham algum hardware disponível, sem necessidade de acesso a internet, usando tanto software livre quanto possível.
## pré-requisitos
<ul>
<li>Qualquer sistema operacional.</li>
<li>Servidor HTTPD, <em>tão somente para uso local</em>, e.g.: <strong>Apache HTTPD Server</strong>.</li>
<li><strong>PHP</strong> script engine provido de PDO para <strong>SQLite</strong>.</li>
<li>Qualquer navegador da web aderente aos padrões <strong>HTML5 + CSS3 + ECMAScript5</strong>.</li>
</ul>
<p>Observação: O esquema do DB é para uso no <strong>SQLite 3.7.13</strong> ou mais recente, acessado pelo <strong>PHP</strong> via PDO.</p>
